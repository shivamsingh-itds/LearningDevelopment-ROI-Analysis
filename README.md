# Learning-Development-L-D-ROI-Analysis-


## 🔍 Overview
This project analyzes the effectiveness of Learning & Development (L&D) programs by measuring ROI and performance improvements across departments, roles, and training programs.

The dashboard provides insights into training impact, helping organizations make data-driven decisions regarding employee development and budget allocation.

## 🚀 Key Features

- 📈 **ROI Trend Analysis** – Track average ROI over multiple years  
- 👥 **Employee Insights** – Analyze data across 3000+ employees  
- 💰 **Cost vs Performance** – Compare program cost with performance gain  
- 🧠 **Program Effectiveness** – Identify top-performing training programs  
- 🏢 **Department Analysis** – Compare performance across departments  
- 📊 **Pre vs Post Training** – Measure improvement after training  

---

## 🛠️ Tools & Technologies

- Tableau
- Data Visualization
- Data Analysis

---

## 📊 Key Insights

- Average ROI is approximately **4.14**, indicating strong training effectiveness  
- Data Analytics and Leadership programs show the highest ROI  
- Post-training performance improved significantly across all departments  
- Finance and HR departments show consistent performance gains  

---

## 📁 Dataset

- Synthetic / Sample L&D dataset  
- Includes employee performance, training programs, costs, and ROI metrics

## 🎯 Objective

To analyze training program effectiveness and help businesses optimize learning investments using data-driven insights.
